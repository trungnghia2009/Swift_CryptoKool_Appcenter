# Swift - CryptoKool

- Support from iOS 13
- Use API from https://www.coingecko.com/vi/api/documentation
- CocoaPod library:
    - ReactiveSwift
    - SDWebImage
    - lottie-ios
- Features:
    - List of top 100
    - Search
    - Crypto detail
